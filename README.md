# 🎓 Student Performance Prediction System

## Project Overview

The Student Performance Prediction System is a Machine Learning project that predicts a student's Performance Index using academic and lifestyle factors. The project uses a Random Forest Regressor model and an interactive Streamlit web application.

## Features

- Predicts student performance
- Interactive Streamlit web application
- Random Forest Regressor model
- User-friendly interface
- Displays predicted Performance Index

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
- Matplotlib
- Seaborn

## Dataset Features

- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced

### Target

- Performance Index

## How to Run

1. Install the required libraries.

```
pip install -r requirements.txt
```

2. Run the model training.

```
python train_model.py
```

3. Run the Streamlit application.

```
streamlit run app.py
```
